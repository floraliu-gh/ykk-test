# LINE Meme Bot - Google Sheet 整合版

這是一個基於 Flask 框架開發的 LINE Bot，旨在提供流暢且具有「沉浸感」的圖片搜尋體驗。本專案核心功能是與 Google Sheets 整合，實現動態更新的梗圖庫搜尋、隨機抽圖及多圖連發功能。目前此機器人架構已具備應對數萬名用戶的穩定性。

## 🌟 核心特色

* **Google Sheet 雲端管理**：圖片資料、關鍵字、集數資訊與音檔連結完全透過 Google 表單管理，修改後自動同步。
* **沉浸感圖片體驗**：捨棄傳統輪播頁面，採用 LINE 原生圖片訊息，點擊即可全螢幕預覽放大，無須跳轉網頁。
* **智慧型多圖連發**：支援單一關鍵字對應多張圖片，並自動計算 LINE API 訊息上限（5 則），確保訊息完美送出。
* **防禦性抓蟲機制**：內建 Try-Except 防護，當資料格式有誤或 API 報錯時會主動提示，避免機器人「已讀不回」。
* **快速回覆按鈕 (Quick Reply)**：提供直覺的操作介面（抽、指令、上一張等），提升使用者互動體驗。
* **效能優化快取**：具備 SHEET\_TTL 快取機制，避免過度抓取 Google 表單，優化回應速度。

## 🛠️ 環境變數設定

部署時請確保已設定以下環境變數：

| 變數名稱 | 說明 |
| :--- | :--- |
| `LINE_CHANNEL_ACCESS_TOKEN` | LINE Developers 提供的 Channel Access Token |
| `LINE_CHANNEL_SECRET` | LINE Developers 提供的 Channel Secret |

## 📊 Google Sheet 欄位結構

請確保您的 Google Sheet 包含以下欄位名稱（第一列）：

1.  **編號**：圖片的唯一數字編號。
2.  **關鍵字**：搜尋時比對的文字。
3.  **圖片網址**：多張網址請以逗號 `,` 或換行隔開。
4.  **集數資訊**：該梗圖的出處或相關文字說明（選填）。
5.  **音檔**：相關聯的音檔網址（選填）。
6.  **藝人**：用於特定藝人搜尋模式（選填）。

> **注意**：建議在表單第一列放置使用說明，程式已設定自動跳過第一筆資料。

## 🚀 部署說明

1.  **複製專案**：將程式碼上傳至 GitHub。
2.  **選擇平台**：推薦使用 Render 或 Heroku 進行部署。
3.  **設定啟動指令**：`gunicorn app:app`。
4.  **設定 Webhook**：將部署後的 URL + `/callback` 填入 LINE Developers 控制台。

## 📖 使用方式

* **關鍵字搜尋**：直接輸入文字（如：`/馬`）即可搜尋相關圖片。
* **數字選圖**：在搜尋列表出現後，輸入對應數字即可查看大圖。
* **隨機抽圖**：輸入 `🎲` 即可在梗圖庫中盲抽一張。
* **藝人模式**：輸入 `/藝人名稱` 可進行特定類別篩選。
* **快速導航**：利用對話框底部的 Quick Reply 按鈕快速執行指令。

## 📦 依賴套件

* `flask`：Web 框架。
* `line-bot-sdk`：LINE Messaging API 整合。
* `requests`：網路請求與表單抓取。
* `gunicorn`：生產環境伺服器啟動器。
